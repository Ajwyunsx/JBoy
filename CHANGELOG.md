# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-XX-XX

### Added
- Initial release
- GBA emulation based on mGBA core
- Material Design 3 cyan theme
- Virtual gamepad with customizable layout
- Game library with grid view
- Save states (4 slots)
- Fast forward (up to 16x)
- OpenGL ES 2.0 video rendering
- OpenSL ES audio output
- Settings screen with video/audio/control options
- ROM file browser
- Dark/Light theme support
